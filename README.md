# Edusym Project

This project aims to create an educational platform using Symfony, API Platform, Vite, React, TypeScript, and ORM Doctrine.

## Backend Technologies

- Symfony 7
- API Platform
- ORM Doctrine
- JWT Auth
- Graphql

## Frontend Technologies

- React 18
- TypeScript
- Vite
- React Redux
- Ant.design components
- Tailwind
- axios
- React Router Dom 6
